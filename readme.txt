== Description ==

An Extension to Easy Member Neo.
Wp-EasyMemberNeo is a wordpress plugin that allows you to protect your posts, ensuring that only your logged in members, with the proper membership level, and drip feed options can view them.

== Installation ==

1. Upload the whole wp-EasyMemberNeo folder to your wp-content/plugins/ folder
2. Go to your Wordpress admin -> Plugins and activate the wp-EasyMemberNeo plugin.
3. Goto Settings -> WP EasyMemberNeo -> Add the URL to your EasyMemberNeo installation & update other options and press Save Changes;
3. That should be it.
